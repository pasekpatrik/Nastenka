# Nástěnka
Aplikace pro rozbrazování rozvrhu a suplování

## Instalace závislostí
```
npm install
```

## Rozběhnutí lokálního server 
Poslouchá na portu 5173
```
npm run client:dev
```

## Sestavění projektu
```
npm run build
```

## Spustění ESlintu
Pro lepší praci doporučuji rozšíření ESlinte - VS Code
```
npm run lint
```
